# 🎂 BirthdayCake

Un pastel de cumpleaños interactivo hecho con **Vue 3 + Vite** donde puedes soplar las velas usando tu micrófono.  
Después de soplar, aparecen globos animados y, tras un tiempo, se muestra una carta especial.  
Diseñado para ser totalmente **responsivo** y funcionar en dispositivos móviles y de escritorio.

---

## ✨ Características

- 🎂 Pastel animado con velas en forma de números.
- 🎤 Detección de soplido usando el micrófono.
- 🎈 Animación de globos al apagar las velas.
- 💌 Carta interactiva que aparece después de un temporizador.
- 📱 Diseño responsive para móviles y PC.
- 🎨 Decoraciones como sprinkles y colores personalizables.

---

## 🚀 Instalación y ejecución

1. **Clona este repositorio**  
   ```bash
   git clone https://github.com/itssinis/BirthdayCake.git
   cd BirthdayCake

2. **Instala las dependencias**
    ```bash
    npm install

3. **Ejectuta el proyecto en modo desarrollo**
    ```bash
    npm run dev
